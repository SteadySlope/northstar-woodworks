# CTR & CONVERSION GUIDELINES

**Status:** Active
**Applies To:** All pages, CTAs, forms, and conversion-focused content

---

## Purpose

Every page on this site has a job: move the visitor closer to taking action. This document defines the rules for CTA buttons, placement strategy, persuasion content, and conversion optimization across the entire site.

---

## 1. CTA Button Rules

### Primary CTA (One Per Page Focus)

Every page must have ONE clear primary action the user should take.

| Element | Rule |
|---------|------|
| **Text** | Action verb + value proposition: "Get Your Free Quote", "Schedule a Consultation" |
| **Color** | Primary brand color (highest contrast on the page) |
| **Size** | Large enough to be unmissable - `lg` or custom large variant |
| **Placement** | Above the fold, repeated in mid-page, and at the bottom |
| **Mobile** | Full-width on mobile (`w-full sm:w-auto`) |

### Secondary CTA

Supporting action that gives an alternative path.

| Element | Rule |
|---------|------|
| **Text** | Softer commitment: "Learn More", "View Our Work", "Call Now" |
| **Color** | Outline or ghost variant (less visual weight than primary) |
| **Placement** | Next to or below primary CTA |

### CTA Text Formulas

**High-converting patterns:**
```
[Action Verb] + [Your/My] + [Desired Outcome]
```

| Strong CTA | Weak CTA |
|-----------|----------|
| "Get Your Free Estimate" | "Submit" |
| "Schedule My Consultation" | "Contact Us" |
| "See Our Work" | "Portfolio" |
| "Call Now - Free Quote" | "Phone" |
| "Start My Project" | "Learn More" |
| "Download the Free Guide" | "Download" |

### CTA Button Rules
- **Never use "Submit"** as CTA text - it tells the user nothing
- **Never use "Click Here"** - vague and bad for accessibility/SEO
- **Use first person where possible** - "Get **My** Quote" converts better than "Get **Your** Quote" (test both)
- **Add urgency when genuine** - "Limited Availability" only if actually true
- **Keep text short** - 2-5 words max for the button itself

## 2. CTA Placement Strategy

### The 3-Touch Rule

Every page should present the CTA at **minimum 3 times**:

```
┌────────────────────────────────────┐
│  HERO SECTION                      │
│  ├─ Primary CTA button             │  ← Touch 1: Above the fold
│  └─ Secondary CTA (optional)       │
├────────────────────────────────────┤
│  CONTENT SECTIONS                  │
│  ├─ Value content...               │
│  ├─ More value...                  │
│  └─ Inline CTA or CTA banner      │  ← Touch 2: After proving value
├────────────────────────────────────┤
│  MORE CONTENT / SOCIAL PROOF       │
│  ├─ Testimonials / reviews         │
│  └─ Trust indicators               │
├────────────────────────────────────┤
│  CTA BANNER / SECTION              │
│  ├─ Compelling headline            │
│  ├─ Brief value prop               │  ← Touch 3: Final push before footer
│  └─ Primary CTA button             │
├────────────────────────────────────┤
│  FOOTER                            │
│  └─ Contact info + CTA link        │
└────────────────────────────────────┘
```

### Placement by Page Type

| Page Type | CTA Placement | Primary CTA |
|-----------|--------------|-------------|
| Homepage | Hero, after services, after testimonials, above footer | "Request a Consultation" |
| Services Hub | Hero, after service cards, above footer | "Request a Consultation" |
| Service Detail | Hero, after features, after process, above footer | "Start Your Custom Piece" / service-specific |
| About Page | After heritage story, after values, above footer | "See Our Work" |
| Gallery | Above grid, below grid, above footer | "Start Your Project" |
| Process | After final step, above footer | "Begin Your Consultation" |
| FAQ Page | Hero, after FAQ sections, above footer | "Still Have Questions? Contact Us" |
| Contact Page | Hero form, phone number prominent, hours section | Form submission |

## 3. Trust & Social Proof Elements

### Trust Indicators (Place Near CTAs)

Trust elements reduce friction. Place them **close to CTAs** where hesitation happens.

| Element | Where to Use | Priority |
|---------|-------------|----------|
| **Star ratings / review count** | Hero, near CTAs, above footer CTA | High |
| **Customer testimonials** | Mid-page, dedicated section | High |
| **Years in business** | Hero stats bar, about page | Medium |
| **Projects completed count** | Hero stats bar, about page | Medium |
| **Certifications / licenses** | Footer, about page, service pages | Medium |
| **Insurance / bonding** | Service pages, footer | Medium |
| **Before/after photos** | Service pages, gallery | Medium |
| **Google/Yelp review badges** | Footer, contact page | Medium |
| **"As Seen In" logos** | Homepage, about (only if real) | Low |
| **Service area map** | Contact page, service area pages | Low |

### Trust Placement Rule

```
CTA Button
├─ Directly below or beside: "★★★★★ 4.9 from 200+ reviews"
├─ Or: "Licensed & Insured | Family Owned Since 20XX"
└─ Or: "Free Estimates - No Obligation"
```

**The trust element answers the question: "Why should I trust this company?"**

## 4. Urgency & Scarcity (Use Ethically)

### Allowed (if genuine)
- "Limited availability this month"
- "Seasonal pricing ends [date]"
- "Book before [season] rush"
- "Free inspection - limited time offer"

### NOT Allowed
- Fake countdown timers
- "Only 2 spots left!" (if not true)
- Manufactured scarcity
- High-pressure manipulation tactics

**Rule:** Only use urgency if it's **actually true.** False urgency destroys trust.

## 5. Content Hierarchy for Conversion

### The AIDA Pattern (Per Page Section Flow)

| Stage | Purpose | Content | Example |
|-------|---------|---------|---------|
| **A**ttention | Stop the scroll | Hero headline, hero image | "Transform Your Outdoor Space" |
| **I**nterest | Create engagement | Pain points, benefits, "did you know" | "Most homeowners overpay by 30%..." |
| **D**esire | Build wanting | Testimonials, before/after, process | "See what we did for the Johnsons..." |
| **A**ction | Get the click | CTA with trust indicator | "Get Your Free Quote ★★★★★" |

### Section Order Template (Service Pages)

```
1. Hero (Attention)
   - Headline with primary keyword
   - Subheadline with value proposition
   - Primary CTA + trust indicator
   - Hero image/background

2. Problem / Pain Points (Interest)
   - What the customer is struggling with
   - Why it matters (consequences of inaction)

3. Solution / Services (Interest → Desire)
   - What you offer
   - How it solves their problem
   - Key features/benefits (use icons)

4. Process / How It Works (Desire)
   - Step-by-step process (3-5 steps)
   - Reduces uncertainty
   - Makes it feel easy

5. Social Proof (Desire)
   - Testimonials with names/photos
   - Star ratings
   - Before/after gallery
   - Stats (years, projects, reviews)

6. FAQ (Desire → Action)
   - Answers remaining objections
   - Removes friction
   - Builds confidence

7. Final CTA Section (Action)
   - Compelling headline
   - Brief value restatement
   - Primary CTA button
   - Trust indicators
   - Phone number
```

## 6. Form Optimization

### Form Rules

| Rule | Detail |
|------|--------|
| **Fewer fields = more conversions** | Only ask for what you need at this stage |
| **Name + Phone + Email minimum** | For quote/contact forms |
| **No required fields that aren't necessary** | Every added field reduces completion rate |
| **Show progress** on multi-step forms | "Step 1 of 3" |
| **Submit button = specific CTA text** | "Get My Free Quote" not "Submit" |
| **Error messages are helpful** | "Please enter a valid phone number" not "Invalid input" |
| **Success state is clear** | Confirmation message + next steps |

### Form Placement
- **Contact/Quote forms:** In the hero (above the fold) on key pages
- **Inline forms:** After a persuasive section mid-page
- **Sticky/floating CTA:** Consider a sticky bar on mobile for key pages
- **Exit intent:** Optional lightbox form (use sparingly)

### Form Field Order
```
1. Name (First + Last, or just First)
2. Phone
3. Email
4. Service needed (dropdown or radio)
5. Message (optional, textarea)
6. [Submit CTA Button]
7. Trust text below: "We'll respond within 24 hours. No spam, ever."
```

## 7. Phone Number Strategy

### Rules
- **Phone number visible in header** on all pages
- **Tap-to-call on mobile** (`<a href="tel:+1XXXXXXXXXX">`)
- **Phone number near every CTA** as an alternative contact method
- **Format consistently** everywhere: (XXX) XXX-XXXX

### Placement
```
Header:    [Logo]  ──────────────  [Phone] [CTA Button]
Hero:      CTA Button  |  or call (XXX) XXX-XXXX
Mid-page:  "Prefer to talk? Call us at (XXX) XXX-XXXX"
Footer:    Phone number in contact column
```

## 9. Measuring Conversion (Analytics Setup)

### Track These Events

| Event | Trigger | Priority |
|-------|---------|----------|
| Form submission | Quote/contact form completed | Critical |
| Phone click | Tap-to-call link clicked | Critical |
| CTA click | Any CTA button clicked | High |
| Scroll depth | 25%, 50%, 75%, 100% | Medium |
| Time on page | 30s, 60s, 120s thresholds | Medium |
| Service page view | Any service page loaded | Medium |
| Blog to service | Blog post → service page navigation | Medium |

### Conversion Rate Benchmarks

| Metric | Good | Great |
|--------|------|-------|
| Contact form submission rate | 2-3% | 5%+ |
| CTA click-through rate | 3-5% | 7%+ |
| Phone click rate (mobile) | 1-2% | 3%+ |
| Bounce rate (service pages) | < 50% | < 35% |

## 10. A/B Testing Priorities

When optimizing, test these elements in order of impact:

1. **CTA button text** (highest impact, easiest to test)
2. **Hero headline** (first thing visitors see)
3. **Form length** (fewer fields vs more qualified leads)
4. **CTA button color/size** (visual prominence)
5. **Social proof placement** (near CTA vs separate section)
6. **Page layout / section order** (which flow converts best)

---

## Rules for Agents

### NEVER:
- Create a page without at least 3 CTA touchpoints
- Use "Submit" or "Click Here" as CTA text
- Place a CTA without a nearby trust indicator
- Create fake urgency or scarcity
- Build forms with unnecessary fields
- Hide the phone number or make it non-clickable on mobile
- Hardcode CTA text (use props so it can be easily changed/tested)

### ALWAYS:
- Follow the 3-Touch Rule for CTA placement
- Use action-verb CTA text with value proposition
- Place trust indicators near CTAs
- Make phone numbers tap-to-call on mobile
- Build forms with the minimum required fields
- Include a compelling final CTA section above the footer
- Make CTAs full-width on mobile for easy tapping
- Define the primary conversion goal per page type
